# Description

Performance stats of company employees application. Four KPI stats are calculated at this moment<br />
Application contains backend Database part, midleware RestApi part and Front-end<br />
Application can work only then all three parts ar deployed and running<br />
<br/>

# Available Scripts

In the project directory, you can run:

### `npm run express-start`

Runs the middleware layer in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to call restApi in the browser.<br/>

### `npm run dev-server`

Runs the front-end layer in the development mode.<br />
Open [http://localhost:3001](http://localhost:3001) to view it in the browser.<br/>
The page will reload if you make edits.<br />

### `npm test`
Unit test is not implemented yet.<br/>

### `npm run build`

Builds the front-end part for production to the `build` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

# Application deployment:

To start midleware layer in folder forbis-kpi-backend execute script:  npm run express-start<br />
To start front-end layer in folder forbis-performance-stat-app execute script:  npm run dev-server<br />
